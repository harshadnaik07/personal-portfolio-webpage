## Project Overview

This project is a personal portfolio website designed and developed to showcase my skills, projects, and professional profile as a Frontend Web Developer. The website serves as a digital resume and an online presence where visitors can learn more about me, explore my technical skills, view my projects, and easily connect with me.

The portfolio is built with a focus on modern UI design, responsiveness, and clean user experience. It uses a minimal yet visually appealing layout with smooth animations, glassmorphism effects, and an interactive design that adapts well across different screen sizes.

## Tools & Technologies Used

This project is built entirely using core web technologies, without relying on any external frameworks or libraries.

### Frontend Technologies

- HTML5
Used to structure the content of the website, including semantic sections such as Home, About, Skills, Projects, and Contact.

- CSS3
Used for styling, layout, animations, and responsiveness. Advanced CSS features like:

   - Flexbox & Grid

   - Media Queries
 
   - CSS Animations & Transitions

   - Glassmorphism UI

   - Gradient backgrounds
are used to create a modern and visually engaging interface.

## Platform & Environment

- Platform: Web Browser

- Development Environment:

  - VS Code (or any code editor)

  - Runs directly in the browser without any build tools

     - Git

## Key Features

- Sticky Navigation Bar
A smooth and modern navbar that stays fixed while scrolling for easy navigation.

- Hero Section
A visually appealing introduction section with a profile image, role description, skill badges, and a call-to-action button.

- About Section
Provides a brief overview of my background, interests, and learning journey in web development.

- Skills Section
Displays technical skills using an interactive grid layout with hover effects.

- Projects Section
Showcases real-world projects such as:

  - JavaScript Quiz Application

  - Real-Time Chat Application

  - Naukari Setu

  - Drowsiness Detection System

- Contact Section
Includes email contact, GitHub profile, LinkedIn profile, and location details.

- Responsive Design
Fully responsive layout that works smoothly on desktops, tablets, and mobile devices.

- Smooth Animations & UI Effects
Subtle hover effects, glow animations, and smooth scrolling improve user experience.
